Instrucciones:
- Default Mode:
	-> Imita la mano. Es el modo por defecto, siempre vuelve a este modo.

- Paper, Rock, Sisors Mode:
	-> Modo de juego aleatorio, se activa manteniendo el meñique estirado y los demás retraidos.

-Win Mode:
	-> Modo de juego en que siempre ganará la maquina, se activa manteniendo el pulgar estirado y los demás retraidos.

-Lose Mode:
	-> Modo de juego en que siempre perderá la maquina, se activa manteniendo el índice estirado y los demás retraidos.


Harware Instructions:
- Pulgar	-> A1 -> Cable Negro	-> Servo 4
- Índice	-> A2 -> Cable Blanco	-> Servo 5
- Corazón	-> A4 -> Cable Azul	-> Servo 6
- Anular	-> A3 -> Cable Rojo	-> Servo 7
- Meñique	-> A5 -> Cable Verde	-> Servo 8
