Robotic Hand
In this Folder you'll find the hole project of the robotic hand made by UAB

-HandAlgoriths:
  Contains both of the algoriths of the hand, the original one presented for the students and the improved one with some details    added and finished

-Hand_Code_COMPLETE:
  Contains only the last version of the code

-RLP_Sprint_Extra (pdf/docx):
  Are the detailed paper of the project made by the students

-Robotic Hand Instructable (pdf/docx):
  The final version of a document showing the actual state of the hand, how to use it, how to fix it, improves, etc
